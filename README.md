Firestrap

A lightweight, feature-rich, and highly customizable bootstrapper for Roblox, inspired by the excellence of Bloxstrap, Froststrap, and Fishstrap.

This launcher is designed to extend your Roblox experience by giving you full control over client settings, performance optimizations, and cosmetic enhancements that the vanilla launcher hides away.

✨ Features

⚡ Performance Engine: Easily unlock frame rates, tweak graphics quality levels, and optimize network settings for lower latency.

🎨 Advanced Customization: Change the in-game font, replace the default mouse cursor, and apply custom Discord Rich Presence (RPC) configurations.

📁 FastFlag Editor: A built-in, user-friendly interface to manage Roblox FastFlags safely without manually editing JSON files.

🛠️ Multi-Instance Support: Run multiple Roblox clients simultaneously out of the box.

🎵 Custom Audio Integration: Bring back the classic "Oof" sound or use your own custom audio files for various client triggers.

🔒 Safe & Open Source: Built with security in mind, interacting strictly with official Roblox APIs and client files.

🚀 Installation

Go to the Releases page.

Download the latest version of the installer (.exe).

Run the installer and follow the on-screen setup wizard.

Launch the application, configure your settings, and click Play.

🛠️ Configuration & FastFlags

This launcher leverages Roblox's native FastFlag system to apply modifications.

⚠️ Disclaimer: While FastFlags are a native feature of the Roblox client, modifying them can sometimes cause stability issues after major Roblox updates. Use the built-in "Reset to Default" option if you experience crashes.
